## 🌟 About The Project

**Jagganath Ji Ki Rasoi** is a beautiful, multi-page website for a local tiffin service and pure vegetarian restaurant. It provides a seamless experience for users to browse the menu, add items to their cart, and place orders directly via **WhatsApp** with auto-filled details.

### ✨ Key Features

- **🛍️ Dynamic Shopping Cart**: Real-time cart updates across all pages using `localStorage`.
- **📱 WhatsApp Integration**: Automatically formats customer details, live location, and order summary into a WhatsApp message.
- **📍 Live Location Capture**: Users can share their exact delivery coordinates with a single click.
- **🚚 Smart Delivery Logic**: Calculates delivery charges automatically (Free within 2-3km, ₹30 otherwise).
- **🖼️ Interactive Gallery**: Beautiful lightbox image gallery for showcasing food.
- **🔗 Clean URLs**: Fully configured for Vercel deployment with clean, extension-less routing (e.g., `/menu` instead of `menu.html`).

## 🚀 How to Run Locally

1. **Clone or Download** the repository to your local machine.
2. Simply double-click on `index.html` to open the website in your default web browser.
3. No build tools or package managers required! It uses pure HTML, CSS, and Vanilla JS.

---

## 🌐 Deployment (Vercel)

This project is pre-configured to be deployed on Vercel.

1. Push the code to a GitHub repository.
2. Go to [Vercel](https://vercel.com) and import the repository.
3. Vercel will automatically read the `vercel.json` file and set up:
   - Clean URLs (removing `.html`)
   - Proper routing for `/home`, `/menu`, `/order`, etc.
